# LLM 모델 선정 벤치마크 (LLM Selection Benchmark)

## 4. 주요 발견

### 4.1 성공 사례 (Bllossom-8B)

**Keyword 질문:**
- "정도전이 사병 혁파를 통해 무력화하려 했던 대상 중 하나인 이성계의 사병 조직의 이름은 무엇인가?"
- **응답**: "정도전이 사병 혁파를 통해 무력화하려 했던 이성계의 사병 조직의 이름은 강철부대 가별초입니다."
- **평가**: 정확한 답변, 간결함, Context 기반

**Abstract 질문:**
- "그... 나라 세울 때 큰 공 세웠던 그 사람이, 왕자들 포함해서 여러 사람들이 가지고 있던 개인 군대를 없애려고 했던 이유가 뭐였어?"
- **응답**: 비공식적 질문 형식 이해, 재상 중심 정치 체제 구축 목적 설명
- **평가**: 비공식적 질문 처리 우수, 정확한 정보 추출

### 4.2 실패 사례

**EEVE-10.8B:**
- **문제**: 영어 응답 생성 (대부분의 응답이 영어)
- **원인**: 프롬프트 처리 또는 모델 설정 문제
- **결과**: 한국어 RAG 시스템에서 사용 불가

**Open-Ko-8B:**
- **문제**: 일부 응답에서 프롬프트 텍스트가 그대로 포함
- **영향**: 응답 품질 저하

## 5. 가설 검증

### 5.1 기각된 가설
- **Small 모델이 Abstract 질문에서 성능 저하**: 반박됨 - Bllossom-8B가 Abstract에서 100% 성공률
- **Medium 모델이 Small보다 우수**: 부분 반박 - Qwen-14B는 성공률 100%이지만 속도가 느림

### 5.2 검증된 가설
- **Keyword 질문은 모든 모델이 비슷한 성능**: 검증됨 - 모든 모델이 100% 성공률

## 6. 결론

**최종 선택: Bllossom-8B**

**선정 근거:**
1. **정량 평가**: 성공률 100%, 평균 지연시간 6.98초 (가장 빠름)
2. **정성 평가**: 정확한 답변, 구조화된 응답, 할루시네이션 없음

4. **실용성**: 빠른 응답 속도로 실시간 서비스에 적합
5. **비용**: 무료 오픈소스, 로컬 배포 가능
